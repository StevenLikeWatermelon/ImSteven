# [ImSteven](http://www.shenwenjie.com/#/index)

基于VUE开发的SPA模式的个人博客。前端webpack+vue+vuex+vue-router,后端语言php。

## Install
``
$ npm run init //编译静态文件
``

``
$ npm run dev //启动本地服务
``

``
注:本地服务仅能看部分前端页面，php源码不开放
``
